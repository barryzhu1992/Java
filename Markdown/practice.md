# Markdown笔记

1. 标题

######六级标题，#后要有一个空格才能生效

2. 字体  
*斜体* ：字体两边一边1个*  
**粗体** ：字体两边一边2个*  
***粗斜体*** ：字体两边一边3个*

4. 脚注  
这是一个链接到谷歌的[^脚注]  
[^脚注]: http://www.google.com  
GitHub不支持脚注
  
5. 代码块  
3个`可以表示代码块  
```
int a = 0;
int b = 3;
print("hello");
a = b;
```  
**4个空格也可以表示代码块**  
  
    int a = 12;
    int b = 17;
    int tem = a;
    a = b;
    b = tem;
